📋 Setup Guide (For Live Mode)
1
Copy .env.example → .env and fill in your LinkedIn email & password.
2
Go to Google Cloud Console → Enable Gmail API → Create OAuth 2.0 credentials → Download as credentials.json in this folder.
3
Place your resume PDF in the resume/ folder as Aditya_Rai_Resume.pdf.
4
Run pip install -r requirements.txt to install dependencies.
5
Click Run Pipeline with Demo Mode OFF for live automation. First run opens a browser for Gmail OAuth authorization.
